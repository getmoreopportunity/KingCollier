# Recurring Patterns - Tasks Repeating

**Purpose:** Spot tasks that repeat 3+ times. These are automation candidates.

---

## How to Use

When you notice a task repeating:
1. Log it here after 2nd occurrence
2. After 3rd occurrence → propose automation
3. If automated → move to WORKFLOWS.md

---

## Template

```markdown
### [Task Name]

**Frequency:** [How often]
**Occurrences:** [Date 1, Date 2, Date 3...]
**Current Process:** [How it's done manually]
**Automation Candidate:** [Yes/No]
**Proposed Solution:** [If yes, what would automate it]
**Status:** [Proposed / Approved / Implemented / Rejected]
```
